Object-Oriented Information Systems

Object-oriented in terms of software is one of many methods used to manipulate tangible things that can be touched, felt, or manipulated. Examples of this is a collection of data and associated behaviour.

Object-oriented analysis

Object-oriented analysis is the process of assessing a problem and identifying objects and the interactions between these objects by focusing on what the proccess entails and the overall outcome required to secure the system.

Object-oriented design

Object-oriented design is the process of documenting the required steps for later implementation. The steps used to define the behaviours of the object focuses on what should be done and the expected output of the process being implemented

Object-oriented programming

Object-oriented programming focuses on the implementation specifics from the docuements design steps of the process in order to produce a working program or system. In terms of security the implementaion focuses on utilising security techniques, methods and tools for the implementaion but in an object-oriented manner.

Details about UML

UML was derived from different notations used in object-oriented methonds. Therefore UML is not a programming language and also not a software development process methdology.

UML is a method used to model complex system. It is an industry modeling technique that combines features from different notations into a single powerful unified language. It aslo provides a mechanism for extension.
